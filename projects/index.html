🤖 Agentic AI Systems

👥 Multi-Agent Workspaces

📝 Aksara
Autonomous multi-agent workspace with sandboxed runtime environments for automated task execution. Technologies: Next.js, FastAPI, Python, Daytona Sandbox, tldraw, Supabase, RabbitMQ, Zep/Graphiti

An AI agent platform that orchestrates specialized digital experts to execute research, document processing, and data analysis. Each agent runs inside an isolated Daytona Linux container, utilizing Playwright for browser automation, Tavily for search retrieval, and RabbitMQ for asynchronous task queueing.

⚔️ Defense & Intelligence

🦅 Project Cartensz
Five-stage threat intelligence pipeline using local classification and LLM synthesis. Technologies: Python, SetFit (NusaBERT), Gemini Flash, DuckDB, Qwen-ASR, FastAPI, Streamlit

A tactical threat analysis pipeline that ingests RSS, Reddit, and YouTube feeds. It uses a locally deployed SetFit (NusaBERT) classifier to filter safe documents at 2,500 texts/sec, forwarding high-risk items to Gemini Flash to compile structured intelligence briefs.

🛡️ ATLAS Multi-Agent Threat Intel
Multi-agent threat detection and tactical intelligence coordinator designed for BNPT and TNI defense operations. Technologies: LangGraph, LiveKit, Neo4j, DuckDB, NeMo Guardrails, FastAPI

An intelligence orchestration system coordinating 7 specialized AI agents using LangGraph. Tracks threat actors and narrative vectors inside a Neo4j knowledge graph, utilizing LiveKit for communication and NeMo Guardrails for data safety and compliance.

🔮 Generative AI & Data Assistants

📊 BPS Papua Chat
Streamlit-based natural language interface to query and visualize provincial statistics. Technologies: Streamlit, Python, Pandas, OpenAI/Gemini, Plotly

A natural language interface for official census datasets from BPS Papua. It parses user questions, queries relevant local Excel/CSV files, performs data aggregation via Pandas, and renders dynamic Plotly visualizations.

🧪 Machine Learning & Signal Processing

🩺 CareCompass Triage
Multi-Task Hierarchical Attention Network (MT-HAN) for clinical specialty triage. Technologies: PyTorch, FastAPI, Streamlit, Docker, Multi-Task Hierarchical Attention (MT-HAN)

A clinical triage classifier using a custom PyTorch Hierarchical Attention Network. Trained from scratch on OSCE clinical dialogues, it achieves 95% specialty routing accuracy, detects patient needs, and uses Monte Carlo Dropout to quantify Bayesian uncertainty.

🫁 Respiratory Diseases Classifier
Acoustic signal classifier for pulmonary disease identification. Technologies: Streamlit, Librosa, Python, Scikit-learn

A web application that extracts acoustic features (mel-spectrograms, chroma, and MFCCs) from respiratory breathing audio recordings (.wav) using Librosa, passing them to a Scikit-learn classifier to detect respiratory conditions.
